## **Simple ant colony simulator with pheromone trails**

![](assets/Screenshot.png)

### Controls:
| Button/Key | Action                       |
|------------|------------------------------|
| WASD       | move camera                  |
| QE         | zoom in/out                  |
| YX         | toggle pheromones            |
| C          | toggle ants                  |
| HJK        | toggle colonies              |
| 123 Space  | Time scale                   |
| PL         | increase/decrease brush size |
| MMB        | place colony                 |
| LMB        | place wall                   |
| LMB + R    | remove wall                  |
| RMB        | place food                   |
| RMB + R    | remove food                  |
