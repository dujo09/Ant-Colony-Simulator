## **Simple ant colony simulator with pheromone trails**

![](assets/ScreenShot.png)

### Controls:
| Button/Key | Action                       |
|------------|------------------------------|
| WASD       | move camera                  |
| QE         | zoom in/out                  |
| YX         | toggle pheromones            |
| C          | toggle ants                  |
| HJK        | toggle colonies              |
| 123 Space  | Time scale                   |
| FV         | Select food/wall             |
| RT         | Select remove/place tool     |
| PL         | increase/decrease brush size |
| RMB        | place colony                 |
| LMB        | place food/wall              |
